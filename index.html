Pour activer les agents et le chat IA avancé, j'ai besoin de ta clé API Anthropic Claude.

Tu peux en créer une sur console.anthropic.com → API Keys → Create Key. Elle commence par sk-ant-.

Sans clé, le chat et les agents fonctionnent en mode offline avec mes réponses préparées. Tu peux configurer plus tard via le bouton ⚙.
